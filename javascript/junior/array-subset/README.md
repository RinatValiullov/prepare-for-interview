## Write a function that checks if one array is a subset of another array.

***

Input 1:
```
const source = [5, 6, 4];
const subset = [4, 5, 6];
arraySubset(source, subset);
```

Output 1: 
```
true
```

Input 2:
```
const source = [5, 5, 5, 4];
const subset = [5, 4, 4];
arraySubset(source, subset);
```

Output 2: 
```
false
```


Input 3:
```
const source = [5, 4];
const subset = [4, 6, 5];
arraySubset(source, subset);
```

Output 3: 
```
false
```
