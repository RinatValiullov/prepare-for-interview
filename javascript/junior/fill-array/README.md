## Write a function `fillArray` that populates a new array with the supplied value.

***

Input 1:
```
const size = 3;
const data = 'hello';
fillArray(size, data);

```

Output 1: 
```
['hello', 'hello', 'hello']
```

Input 2:
```
const size = 4;
const data = { prop: 'x' };
fillArray(size, data);

```

Output 2: 
```
[{ prop: 'x' }, { prop: 'x' }, { prop: 'x' }, { prop: 'x' }]
```
