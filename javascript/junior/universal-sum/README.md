## Write a function that takes two numbers and returns their sum. The function can be called like below:

```
sum(11, 19); // 30
sum(11)(19); // 30
sum(11)()(19); // 30
sum(11)()()()(19); // 30
sum(11)()()()()()()()()()(19); // 30
sum()()()(11)(19); // 30
sum()()(11)()(19); // 30
sum()(11)()(19); // 30
sum()()()()(11)()()()()(19); // 30
```
